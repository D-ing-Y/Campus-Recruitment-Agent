## 12. 明确禁止

- 不用 mock 替代 v0.4 文档读取、证据写入和 checkpoint 的最终集成实现。
- 不让 checkpoint、聊天摘要或 LLM 输出绕过 Evidence Store 成为画像事实源。
- 不让 LangGraph State 保存完整 PDF、长文本或二进制。
- 不让 LLM 自行突破预算、决定不可枚举动作或覆盖历史 Claim。
- 不在 v0.4 同时加入岗位爬虫、RAG、分布式存储或 Multi-Agent。
- 不删除旧 Runtime、SearchGoal 和历史版本测试。
