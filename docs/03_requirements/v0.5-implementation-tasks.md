## 14. 明确禁止

- 不在解析或模型总结前跳过原始响应归档。
- 不让社区面经确认岗位存在、硬性资格或截止时间。
- 不让 LLM 直接合并重复岗位、计算分母或修改统计阈值。
- 不根据 CandidateProfile 能力静默过滤岗位。
- 不把 Cookie/cURL/Authorization 写入 State、trace、Artifact metadata、报告或 Git。
- 不绕过验证码、登录、限流、robots 或来源条款。
- 不在 v0.5 引入匹配评分、RAG、分布式存储或 Multi-Agent。
