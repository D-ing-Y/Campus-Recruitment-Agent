## 1. 范围

本轮验证代码实现前的来源可行性：

1. BOSS 登录态能否完成最小岗位搜索与详情读取；
2. 牛客登录态能否完成最小经验搜索与详情读取；
3. 第三方候选岗位能否路由到企业官网并形成明确核验状态。

这不是 SourceAdapter 代码验收。raw-before-parse、credential service、分页、重试、
SourceRunReceipt 和 Graph interrupt/resume 必须在实现后通过独立的 adapter live smoke。

## 4. 结论与限制

三个实现前 P1 浏览器来源 smoke 均完成，v0.5 可以进入代码实现。

验证通过的是“来源链可访问、可搜索、可读取、可核验”；尚未通过的是“本项目
SourceAdapter 已实现并能 raw-before-parse”。代码完成后仍必须执行：

1. `zhaopin_jobs`、`nowcoder_experience`、`official_careers` 三个 adapter opt-in live smoke；
2. 原始响应先归档，再解析和投影；
3. 不含凭据的 SourceRunReceipt；
4. auth interrupt/resume、分页、限流、幂等和凭据脱敏；
5. 至少一个 confirmed/ambiguous/not-found 的自动化官网核验案例。
