## 失败案例

- 不存在的 `evidence_fragment_id`：拒绝写入并返回可读错误。
- 引用当前证据集之外的 Artifact：拒绝写入。
- LLM 非法 JSON/schema：有限重试；provider error 转换为结构化失败记录。
- SQLite 写入失败：本次新建 blob 被清理，不留下失效 URI。

## 结论

v0.3 验收通过。统一证据层与画像契约可作为 v0.4 Candidate Profile LangGraph 的稳定基础。当前不包含真实爬虫、RAG、分布式存储、Multi-Agent 或复杂 PDF 版式解析。
