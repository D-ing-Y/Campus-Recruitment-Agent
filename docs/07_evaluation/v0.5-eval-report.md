## 4. 覆盖的关键案例

- 三类 fixture adapter 使用同一 raw-first 归档路径，支持 query、cursor、pagination 和 batch 幂等。
- `zhaopin_jobs`、`official_careers`、`nowcoder_experience` live adapter 默认关闭；未显式启用时返回 `policy_blocked`。
- 成功页及登录、限流、404、结构变化等失败响应均先归档；raw Blob/Artifact 写入失败时不创建成功 SourceDocument，解析器不会运行。
- 官网解析链覆盖 JSON-LD、已注册结构化 adapter hook、确定性文本、strict LLM hook 和 `adapter_required`。
- 官网 redirect/domain budget 有 allowlist 校验；声明式 AdapterSpec 不能扩大域名或页面预算。
- 京东科技 `AI Agent开发工程师` 脱敏智联 fixture 得到 `official_not_found`，第三方岗位继续保留。
- 牛客脱敏面经只生成 interview/project/tech-stack signal，不支持岗位存在、学历、地点、薪资或截止时间。
- 三个去重岗位、两家公司形成 sufficient 岗位族；小样本最多为 `insufficient_sample`。
- 授权中断只传 `credential_ref`；authorized、skip 和 SQLite 重启恢复均通过。
- run exporter 生成 search scope、user needs、query history、source receipts/index、normalized JSONL、identity/resolution 和 Markdown report。
