# Project Charter

## 项目目标

构建一个基于 LangGraph 的垂直领域 Job Intelligence Agent，用于秋招岗位情报收集、证据归档、岗位需求分析、面试经验总结、最小能力包生成和项目策略建议。

## 当前原则

- 文档先行，代码跟随文档。
- 先做单 Agent 最小闭环，再做工具、memory、RAG 和多 Agent。
- 使用 API 调用大模型，不训练基础大模型。
- 所有结论尽可能绑定证据。
- 不绕过网站验证码或反爬验证。
- Mac 端作为主要交互和设计窗口，后续云服务器作为部署环境。

## 非目标

- 不复刻完整 Codex 或 DeerFlow 平台。
- v0.1 不接入真实招聘网站。
- v0.1 不做 RAG、向量数据库、多 Agent、完整前端或服务器部署。
